# Procurement Spend Analysis

## Project Overview
This project analyzes procurement spend data to identify spending patterns, supplier performance, and cost optimization opportunities. The analysis focuses on supplier spend, category spend, monthly trends, and purchasing behavior.

## Objectives
- Calculate total procurement spend
- Identify top suppliers by spend
- Analyze spend per category
- Analyze monthly spending trends
- Identify top buyers
- Identify most purchased items
- Provide insights for cost optimization

## Tools Used
- SQL (Databricks)
- Excel
- Power BI / Looker Studio
- GitHub
- Miro (Flowchart)
- Gantt Chart (Project Planning)

## Project Steps
1. Project Planning
2. Data Collection
3. Data Cleaning
4. Data Processing using SQL
5. Data Analysis
6. Data Visualization
7. Dashboard Creation
8. Presentation
9. Documentation
